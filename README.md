# 精美名片生成器

一个现代化的在线名片生成工具，帮助您快速制作专业的个人名片。

## 功能特点

- 🎨 **精美设计**：深色渐变背景，专业商务风格
- 📸 **头像上传**：支持图片上传和裁剪调整
- ⚡ **实时预览**：输入信息即时更新预览效果
- 💾 **本地下载**：生成高质量JPG格式名片
- 📱 **响应式设计**：完美适配移动端和桌面端
- 🎯 **标准尺寸**：3.5英寸 × 2英寸标准名片尺寸

## 使用方法

1. **打开网页**：在浏览器中打开 `index.html`

2. **填写信息**：
   - 点击上传头像，选择图片后可裁剪调整
   - 填写姓名、职务、公司等基本信息
   - 添加个人标语/Slogan
   - 输入联系方式（电话、邮箱、地址等）

3. **预览效果**：右侧实时显示名片效果

4. **下载名片**：点击"下载名片"按钮保存JPG格式文件

## 技术栈

- **HTML5**：页面结构
- **CSS3**：响应式设计和动画效果
- **JavaScript**：交互功能和Canvas生成
- **Cropper.js**：图片裁剪功能
- **html2canvas**：将DOM转换为图片

## 名片信息字段

- 头像（支持裁剪）
- 姓名
- 公司职务
- 公司名称
- 公司网址
- 个人标语/Slogan
- 电话
- 邮箱
- 公司地址

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

- 头像图片支持 JPG、PNG 等常见格式
- 图片大小建议不超过 5MB
- 生成的名片尺寸为 3.5" × 2" (标准名片尺寸)
- 移动端自动切换为上下布局
